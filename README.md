# Merchant Payment System

A comprehensive smart contract solution for small business payment processing with built-in fraud detection, chargeback management, and instant settlement capabilities.

## Key Features
- **Merchant Registration**: Businesses can register and manage their payment processing accounts
- **Transaction Processing**: Secure payment processing with automatic fraud detection
- **Chargeback Management**: Built-in dispute resolution system with time-bound windows
- **Fraud Detection**: Automated risk assessment based on transaction patterns and amounts
